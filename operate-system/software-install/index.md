# Linux 系统软件安装

欢迎来到 Linux 系统软件安装指南。本节包含了各种 Linux 系统软件的安装和配置教程，帮助您快速部署和使用这些软件。

## 可用软件

目前本节包含以下软件的安装指南：

### Slurm 工作负载管理器

[Slurm](./linux/slurm/) 是一个开源的、容错的、高度可扩展的集群管理和作业调度系统，适用于大型和小型 Linux 集群。

- [单机版安装指南 (Rocky 9)](./linux/slurm/standalone)：适合个人学习和小型测试环境的单机版 Slurm 安装指南。
