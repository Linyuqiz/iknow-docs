# Slurm 工作负载管理器

欢迎来到 iKnow Docs 的 Slurm 部分。本节包含关于 Slurm 工作负载管理器的安装、配置和使用指南。

## 什么是 Slurm？

Slurm（Simple Linux Utility for Resource Management）是一个开源的、容错的、高度可扩展的集群管理和作业调度系统，适用于大型和小型 Linux 集群。Slurm 提供了三个关键功能：

1. 为用户提供对计算资源的访问权限
2. 提供框架来启动、执行和监控工作
3. 管理竞争资源的争用

## 主要特点

- **可扩展性**：可以扩展到数千个节点的集群
- **容错性**：设计为在各种故障情况下保持运行
- **高度可配置**：提供了大量配置选项
- **安全性**：支持多种身份验证机制
- **开源**：基于 GPL 许可证发布
